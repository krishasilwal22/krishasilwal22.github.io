Problem Statement

Traditional input devices like a mouse and keyboard are not accessible to everyone, particularly individuals with limited fine motor control or physical impairments. Gesture-based systems offer an alternative, but many applications still require precise hand movements, making them difficult to use for many people.

While creating this project, we were interested in exploring how hand tracking technology could allow users to interact with digital objects more intuitively. Our goal was to develop a system where users could move and stack virtual blocks using only their hands, without needing a mouse or keyboard.

What We Built & How It Works

We developed a real-time hand tracking system that detects hand movements through a webcam and allows users to interact with blocks on the screen. If a hand moves close to a block, it "grabs" the block, allowing it to be moved. When stacked, the blocks behave as if they are real, staying in place unless moved again.

Technologies We Used

  • Python – To program the logic behind object movement.
  • OpenCV – For capturing video and detecting hand positions.
  • cvzone – To simplify hand tracking and gesture-based interactions.

Why This Matters & Potential Applications

  • Accessibility: Could be helpful for users who find traditional input devices challenging.
  • Potential Uses: Could be expanded for gesture-controlled games, AR/VR applications, and assistive technology.
  • Hands-Free Interaction: Demonstrates how computer vision can create intuitive digital experiences without physical controllers.

What Could Be Improved?

  • Improve tracking accuracy using AI-based gesture recognition.
  • Integrate voice control for users with limited hand mobility.
  • Explore alternative inputs like eye tracking or adaptive gestures.
  • Refine stacking logic for a smoother user experience.
