Technologies Used

Python Code


from cvzone.HandTrackingModule import HandDetector
import cv2

cap = cv2.VideoCapture(0)
detector = HandDetector(detectionCon=0.8, maxHands=2)

while True:
    success, img = cap.read()
    hands, img = detector.findHands(img)
    cv2.imshow("Image", img)
    if cv2.waitKey(1) == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()

Analyzing the Gender Pay Gap Using Regression Modeling

Problem Statement

The gender pay gap is a widely discussed issue, but what factors actually contribute to it? This project aimed to predict the gender pay gap across 25 U.S. states by analyzing minimum wage, unemployment rate, women’s median earnings, labor force participation, and pay transparency laws.

What We Did

Using Excel, we performed multiple linear regression to examine how these economic and policy factors relate to the gender pay gap. Minimum wage emerged as the strongest predictor, explaining 27.87% of the variation in pay disparities.

Key Findings

  • Lower minimum wages correlate with wider pay gaps.
  • Median earnings for women were skewed, meaning a few high earners may distort pay equity perceptions.
  • Unemployment and labor force participation were not strongly linked to wage disparities.
